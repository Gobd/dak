# Kasa Controller

Dedicated app for controlling TP-Link Kasa smart plugs with scheduling and energy monitoring.

## Features

- List all connected Kasa devices
- Toggle device power on/off
- Brightness control for dimmable devices
- Energy usage and power monitoring
- Schedule management (recurring, sunrise/sunset-based)
- Countdown timers
- Energy history
- Device status indicators

## Development

```bash
pnpm dev      # Start dev server on port 5176
```

Requires home-relay running with Kasa device access.
