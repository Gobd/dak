# Health Tracker

Family health management app for tracking medications, shots, and medical history.

## Features

- Shot schedules with intervals and dosages
- Medicine courses with multi-day durations
- As-needed (PRN) medication tracking
- Multiple family members
- Medication history and logging
- Real-time sync across devices

## Development

```bash
pnpm dev      # Start dev server on port 5173
```

Requires Supabase project with auth and database configured.
