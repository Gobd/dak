# Family Chores

Chore management and reward system for families with points, leaderboards, and flexible scheduling.

## Features

- Flexible scheduling (daily, weekly, monthly, every X days, goal-based)
- Point-based reward system
- Leaderboard with member rankings
- Weekly and daily task views
- Goal/habit tracking with target completions
- Parent PIN protection for settings
- Multi-instance tasks (same task multiple times per day)
- Member management with emoji avatars
- Onboarding flow for new families
- Real-time sync across devices

## Development

```bash
pnpm dev      # Start dev server on port 5174
```

Requires Supabase project with auth and database configured.
